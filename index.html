Metric/Imperial Converter

Just insert value and unit then convert

How To Use

  • It support gal-L,lbs-kg and mi-km convertion and vice versa
  • You should use mentioned units.Lowercase or uppercase doesnt matter
  • You can use fractions, decimals or both in value (ie. 5, 1/2, 2.5/6)
  • if no value is provided it will default to 1
  • Result is json object that contains init value and output value
